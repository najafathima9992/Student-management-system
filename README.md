# 🎓 Student Management System (Flask + SQLite)

## 📌 Overview

This is a **full-stack web application** built using Python Flask and SQLite that allows users to manage student records efficiently.
The application supports complete **CRUD operations** (Create, Read, Update, Delete) with a simple and responsive UI.

---

## 🚀 Features

* ➕ Add new students
* 📋 View all students
* ✏️ Edit student details
* ❌ Delete student records
* 🔍 Search students by name
* ✅ Form validation
* 🎨 Responsive UI using Bootstrap

---

## 🛠️ Tech Stack

* **Backend:** Python (Flask)
* **Frontend:** HTML, Bootstrap
* **Database:** SQLite
* **Templating Engine:** Jinja2

---

## 📁 Project Structure

```
student-app/
│── app.py
│── students.db
│── templates/
│     ├── index.html
│     ├── add.html
│     ├── edit.html
```
---

## 🧠 How It Works

* Flask handles routing and backend logic
* SQLite stores student data
* HTML templates render dynamic data using Jinja2
* Forms send data using POST requests
* Data is fetched and displayed using GET requests

---

## 🔄 CRUD Operations

| Operation | Description          |
| --------- | -------------------- |
| Create    | Add new student      |
| Read      | View student list    |
| Update    | Edit student details |
| Delete    | Remove student       |
